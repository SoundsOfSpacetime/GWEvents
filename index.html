Plots are pre-generated in Python and exported to JavaScript. The parameter values corresponding to highest log_likelihood of SEOBNRv4PHM posterior samples from The Gravitational Wave Open Science Center (GWOSC). Plots were constructed using LALSimulation's XLALSimInspiralChooseTDWaveform() waveform generator. The starting frequency for each event is set to 20 Hz, so the beginning of sounds may not be audible on some devices. Headphones and Google Chrome are recommended.
